Project Overview

nSMM is an open-source project that brings the iconic Super Mario Maker experience to TI-Nspire calculators, PC and the web. Design levels with precise control, test them instantly, and share with friends!

Key Features:

  • Intuitive level editor
  • Cross-platform: calc & browser
  • No Ndless needed
  • Open-source and community-driven
